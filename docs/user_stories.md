# User Stories

## Support Ops

- As a Support Operations Manager, I want tickets auto-classified by issue type so that I can reduce manual queue review.
- As a Support Operations Manager, I want weekly summaries of recurring themes by segment so that I can brief leadership quickly.
- As a Support Operations Manager, I want to compare escalation rates and resolution time across categories so that I can redesign workflows.

## Product

- As a Product Manager, I want to rank product pain points by business impact so that roadmap prioritization is evidence-based.
- As a Product Manager, I want to see which issues are concentrated among enterprise or high-ARR accounts so that strategic customer pain is not hidden.
- As a Product Manager, I want support pain linked to feature request demand so that I can tell the difference between bug cleanup and product opportunity.

## Customer Success

- As a Customer Success Lead, I want to identify which segments have the highest churn-risk ticket rate so that I can deploy proactive outreach.
- As a Customer Success Lead, I want to see whether onboarding or support experiments improved outcomes so that I can scale the right playbooks.

## Executive / Leadership

- As an executive stakeholder, I want an overview of ticket volume, CSAT, escalation rate, and renewal-risk indicators so that I can understand support health at a glance.
- As an executive stakeholder, I want concrete recommendations with data backing so that I can decide what to fund next.
